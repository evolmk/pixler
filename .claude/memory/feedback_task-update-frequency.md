Don't waste tokens updating task progress every few seconds. Update sparingly — every ~5 minutes during long-running work, or batch by sprint/phase.

**Why:** Token efficiency matters; frequent status updates are noise.

**How to apply:** When completing a full milestone plan, update `_docs/plans/MILESTONE-TRACKER.md` — check off the milestone, update "Last Plan Completed" and "Next Plan" fields.
